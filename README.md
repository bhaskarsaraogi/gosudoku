gosudoku is a command line util to generate sudoku puzzle, solve sudoku or both

## Installation

```
$ go get github.com/bhaskarsaraogi/gosudoku/cmd/gosudoku
```

## Play

#### Generate a sudoku puzzle:
```
$ ./gosudoku -generate
```
